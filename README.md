# Draw by finger 🎨
노트북 카메라를 통해 출력되는 실시간 영상에 그림을 그리는 프로그램이다. </br>
노트북 화면을 터치하여 그리는 것이 아닌 손동작을 이용하여 허공에 그림을 그리는 것이 특징이다. <br></br>
